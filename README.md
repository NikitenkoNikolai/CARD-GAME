# CARD-GAME

Карточная игра «Детство» на PyGame с архитектурой MVC и чистым ООП.

---

## Стек

- Python 3.10+
- PyGame 2.5+
- SQLite

---

## Архитектура

### Паттерн MVC

| Компонент   | Файл                                     | Ответственность                     |
|-------------|------------------------------------------|-------------------------------------|
| **Model**   | `process_explorer.py`, `data_base.py`, `game_interface.py` | Состояние игры, сохранения, логика ходов |
| **View**    | `View.py`                                | Отрисовка интерфейса, карт, врагов  |
| **Controller** | `Controller.py`                       | Обработка событий, управление потоком игры |

### Иерархия карт (наследование + полиморфизм)

Card 
├── Stick # урон 15, стоимость 15 маны
├── Stone # урон 10, стоимость 10 маны
├── Cry # бафф брони +15, стоимость 10 маны
├── Cunning # обмен бронёй/здоровьем, стоимость 20 маны
├── Cuteness # бафф здоровья +10, стоимость 10 маны
├── Berserk # урон по сегменту руки + самоповреждение, стоимость 25 маны
└── Prophet # копирование карты противника через SegmentTree, стоимость 15 маны
